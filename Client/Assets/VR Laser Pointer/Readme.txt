
Как использовать

1. Кинуть в правую или левую руку префаб "VR Canvas Laser Pointer"
2. Потавить на сцену префаб "EventSystem + VR Input Module"

3. На префабе "VR Canvas Laser Pointer" в Инспекторе в поле camera перетянуть лазер поинтер
4. На лазер поинтере в Инспекторе в поле Event System Input Module перетянуть "EventSystem + VR Input Module" со сцены
5. На всех мировых канвасах поставить в поле Event Camera лазер поинтер
6. Удалить со сцены все остальные Event System, кроме того, что на префабе "EventSystem + VR Input Module"
