
Как использовать:
1. Поместить 'Emoji Manager' на сцену. Можно поставить галочку Don'tDestroyOnLoad, если планируется переход со сцены на сцену.
2. Вызывать публичный статичный метод CreateEmoji() из класса EmojiManager или из класса Emoji
Эмоция будет уничтожена автоматически после проигрывания анимации.

UI.
В папке с эмоциями есть префаб UI панельки с кнопками для эмоций. Но их нужно привязать к какому-нибудь коду через Инспектор.

Передача эмоции по сети:
1. Вызвать какую-то эмоцию на локальном персонаже.
2. Передать по сети значение int
3. На удалённом персонаже привести int к типу Emoji.Type и вызвать метод CreateEmoji()
